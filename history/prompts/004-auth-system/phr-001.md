## User Request

```
Integrate Supabase authentication into the frontend. Create sign in, sign up, and profile setup pages. Personalize the chatbot based on user profiles (programming knowledge, learning goals, robotics experience).
```

## Agent Response

Implemented Supabase Auth with:
- AuthContext provider for global auth state
- Sign in/up pages using Supabase Auth UI components
- Profile setup page for collecting learning preferences
- Navbar auth UI with dropdown (avatar, name, logout)
- Chatbot personalization with user profile data
- Backend auth routes for signup/signin
