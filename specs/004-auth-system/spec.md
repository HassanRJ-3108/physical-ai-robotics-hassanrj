# Authentication System

## Spec

Implement Supabase Auth for user authentication and chatbot personalization.

**Goals:**
- Email/password signup & signin
- User profile data (learning preferences)
- Personalized chatbot responses
- Navbar auth UI with dropdown

## Plan

1. **Supabase**: Create project, user_profiles table, RLS policies
2. **Frontend**: AuthContext, signin/signup pages, profile setup
3. **Navbar**: Auth UI with avatar + dropdown menu
4. **Chatbot**: Personalize with user data, send auth tokens
5. **Backend**: Validate JWT tokens, fetch profiles

**Tech:** Supabase Auth, Auth UI React, JWT tokens
